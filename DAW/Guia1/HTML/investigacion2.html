Datos NULL y Undefined

Que es un dato NULL y Undefined

NULL. Las variables tipo null son aquellas que se les define el valor de "No valor"

Undefined. Las variables Undefined son aquellas que no tienen definido ningun valor

Diferencias

La diferencia principal entre uno y otro es que uno existe y otro no, lo que nos quiere decir que el valor null, JavaScript lo entiende como un dato que no tiene valor, por otro lado a undefined para Javascript no existe.

Usos

Null es mayormente utilizado para comparar y una cadena o una variable tiene o no valor, por lo que podemos decir que se usa como un comprobador o verificador; En cambio, un dato undefined puede ser utilizado como referencia
